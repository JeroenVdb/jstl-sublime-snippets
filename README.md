jstl-sublime-snippets
=====================

JSTL Tag support in Sublime Text 2 via snippets.

##Install
- Navigate to /Packages/User/ folder
- Create "Jsp" folder if necessary
- Clone this repository in the "Jsp" folder

##Usage
Hit c: tab to display tags.

##Supported tags
#Core taglib
- set
- choose
- if
- out
- param
- url

#PE taglib
- staticUrl
- developmentInfo

#Spring taglib
- message
